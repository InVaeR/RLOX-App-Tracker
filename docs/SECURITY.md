# Безопасность

## Обновления

- Все сетевые запросы только по HTTPS
- SHA-256 установщика проверяется перед запуском
- При скачивании используется временный `.part` файл
- Размер загрузки ограничен

## Хранение данных

- БД SQLite хранится в `%LOCALAPPDATA%` (не синхронизируется)
- Заголовки окон сохраняются только с согласия пользователя
- Логи ротируются (2 MB × 3 файла)

## Приватность

По умолчанию сохранение заголовков окон включено.
Рекомендуется отключить эту опцию при работе с конфиденциальными данными.
