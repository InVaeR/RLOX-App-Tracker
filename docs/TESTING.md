# Тестирование

## Запуск тестов

```bash
python -m pytest app/tests/ -v
```

## Тесты трекинга

- Обычный активный тик
- Фоновое приложение
- Idle выше порога
- Пауза короче 10 секунд
- Длинная пауза
- Сон компьютера
- Изменение системного времени назад
- Смена суток
- Удаление приложения с активной сессией
- Аварийное восстановление незакрытой сессии
- Накопление дробных секунд при интервале 1,5
- Переполнение Windows tick counter
- Сессия, пересекающая границу дня
- Сортировка длительности

## Тесты репозитория

- CRUD операции с watched_apps
- CRUD операции с sessions
- Агрегация статистики
- Очистка данных

## Тесты миграции

- Отсутствие старых данных
- Миграция при наличии старых данных
- Пропуск при существующей новой БД
- Пропуск при наличии маркера
