# Миграция из RusLOXPy v1.x

При первом запуске RLOX App Tracker v2.x автоматически проверяет наличие данных
в `%APPDATA%\RusLOXPy\` и переносит их в `%LOCALAPPDATA%\RLOX App Tracker\`.

## Что переносится

- `tracker.db` — вся история отслеживания
- `config.json` — настройки приложения

## Процесс

1. Проверяется маркер завершённой миграции
2. Проверяется наличие новой БД
3. Выполняется SQLite checkpoint старой БД
4. Создаётся резервная копия
5. Копируются файлы
6. Записывается migration report

## Безопасность

- Старые данные **не удаляются**
- Миграция выполняется однократно
- При ошибке создаётся пометка в migration report
