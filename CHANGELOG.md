# Changelog

## [1.1.0] - 2026-07-12
### Added
- Экспорт статистики в CSV (кнопка на дашборде)
- Живой тултип трея (показывает текущее приложение и время)
- Горячие клавиши: Ctrl+1/2/3 для навигации, Delete для удаления в списке
- Double-click для выбора процесса в диалоге добавления
- Кнопка «Обновить список» в диалоге добавления приложения
- Кнопка «Добавить приложение» в пустом состоянии дашборда
### Changed
- Проверка обновлений по кнопке — асинхронная (не блокирует UI)
- Сохранение настроек без модального окна (кнопка «✓ Сохранено» на 1.5 с)
- Статусбар обновляется раз в 5 секунд (было 1 с), без времени и прямых запросов БД
- График и легенда скрыты, пока нет данных
### Fixed
- Удалён мёртвый импорт в app_icons.py
- Год в LICENSE синхронизирован

## [1.0.1] - 2026-07-12
### Fixed
- Исправлена версия приложения и ссылки на репозиторий
- Проверка обновлений вынесена в фоновый поток (не блокирует UI)
### Added
- Опция «Проверять обновления при запуске» в настройках

## [1.0.0] - 2026-07-12
### Added
- Отслеживание времени выбранных приложений (активное/фоновое)
- Дашборд со статистикой (день/неделя/месяц/всё время)
- Список отслеживаемых приложений с иконками
- Работа в трее, автозапуск, пауза
- Проверка обновлений через GitHub Releases
- Single-instance, восстановление сессий после краха
